# FlashStudy

FlashStudy is a small, friendly iOS app for working with flashcards — built as a personal project for fun and shared here on GitHub. This is the first version and focuses on the essentials to keep things simple and usable.

Key features
- Add flashcards: create simple front/back pairs and store them locally.
- Review flashcards: flip and browse cards to train memory and track progress.
- Theme switch: light and dark themes are available — I personally prefer dark, so a theme option is included.

Who this is for
- Students and anyone preparing short facts or vocabulary.
- People who want a minimal, distraction-free tool for memorization.

Getting started
1. Open the project in Xcode: open `FlashStudy.xcodeproj` or `FlashStudy.xcworkspace`.
2. Select a simulator or device and press Run (⌘R).

# FlashStudy 

FlashStudy — простое и дружелюбное iOS‑приложение для работы с карточками, созданное мною как личный проект для удовольствия и размещённое на GitHub. Это первая версия, которая сосредоточена на базовой функциональности, чтобы всё было просто и удобно.

Основные возможности
- Добавление карточек: создавайте пары «вопрос — ответ» и сохраняйте их локально.
- Просмотр карточек: переворачивайте и пролистывайте карточки для тренировки памяти.
- Смена темы: доступна светлая и тёмная тема — мне лично больше нравится тёмная, поэтому такая опция добавлена.

Кому это может быть полезно
- Студентам и тем, кто готовит материал для запоминания.
- Тем, кто предпочитает минималистичные инструменты без лишних отвлекающих элементов.

Как запустить
1. Откройте проект в Xcode: `FlashStudy.xcodeproj` или `FlashStudy.xcworkspace`.
2. Выберите симулятор или устройство и нажмите Run (⌘R).
